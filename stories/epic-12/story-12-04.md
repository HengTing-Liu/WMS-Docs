# Story 12-04 库存统计看板

## 0. 基本信息

- Epic：报表与统计
- Story ID：12-04
- 优先级：P1
- 状态：Draft
- 预计工期：1d
- 依赖 Story：Epic-02（库存查询）
- 关联迭代：Sprint 5

---

## 1. 目标

实现库存统计看板，为管理层提供仓储核心指标的实时数据展示。

---

## 2. 业务背景

库存统计看板聚合关键业务指标，提供一目了然的数据视图，辅助管理决策。

---

## 3. 范围

### 3.1 本 Story 包含

- 库存统计卡片（总库存量、总SKU数、库存金额等）
- 库存分布统计（按仓库、按物料分类）
- 出入库趋势图（近7天/30天）
- 效期预警统计
- 呆滞物料统计

### 3.2 本 Story 不包含

- 自定义报表配置（后续迭代）

---

## 4. API / 接口契约

### 4.1 统计卡片接口

**接口**：GET `/api/dashboard/inventory/stats`

**响应**：

```json
{
  "code": 200,
  "data": {
    "totalQuantity": 1000000,
    "totalSku": 5000,
    "totalAmount": 50000000.00,
    "frozenQuantity": 50000,
    "availableQuantity": 950000,
    "lowStockCount": 100,
    "nearExpireCount": 50,
    "stagnantCount": 30
  }
}
```

### 4.2 库存分布接口

**接口**：GET `/api/dashboard/inventory/distribution`

**响应**：

```json
{
  "code": 200,
  "data": {
    "byWarehouse": [
      {"name": "上海仓", "value": 500000},
      {"name": "北京仓", "value": 300000},
      {"name": "广州仓", "value": 200000}
    ],
    "byCategory": [
      {"name": "试剂", "value": 400000},
      {"name": "耗材", "value": 350000},
      {"name": "设备", "value": 250000}
    ]
  }
}
```

### 4.3 出入库趋势接口

**接口**：GET `/api/dashboard/inbound-outbound/trend`

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|---:|------|
| days | int | N | 天数（默认7） |

**响应**：

```json
{
  "code": 200,
  "data": {
    "dates": ["2026-04-01", "2026-04-02", "2026-04-03"],
    "inbound": [1000, 1200, 800],
    "outbound": [900, 1100, 950]
  }
}
```

### 4.4 预警统计接口

**接口**：GET `/api/dashboard/inventory/warning`

**响应**：

```json
{
  "code": 200,
  "data": {
    "lowStock": [
      {"materialCode": "MAT001", "materialName": "试剂盒", "currentQty": 10, "minQty": 50}
    ],
    "nearExpire": [
      {"materialCode": "MAT002", "materialName": "培养基", "expireDate": "2026-04-15", "daysLeft": 12}
    ],
    "stagnant": [
      {"materialCode": "MAT003", "materialName": "移液管", "lastMoveDate": "2026-01-01", "daysNoMove": 92}
    ]
  }
}
```
