## 一、系统概述

WMS（Warehouse Management System）仓库物流系统是公司自研的仓储全流程管理系统，涵盖库存查询、入库、出库、库存调整、质控、提货单、出库单、中转仓、外贸发货、运单配送十大核心模块。

## 八、安全架构

### 8.1 认证授权

- **认证方式**: JWT Token
- **会话管理**: Redis 分布式会话
- **权限控制**: RBAC（基于角色的访问控制）
- **数据权限**: 行级数据权限（租户隔离 + 仓库权限）

### 8.2 安全措施

| 安全项 | 实现方案 |
|--------|----------|
| 敏感数据 | 加密存储（手机号、身份证等） |
| 日志审计 | 操作日志全记录（谁、何时、做了什么） |
| SQL注入 | MyBatis参数绑定 |
| XSS攻击 | 输入校验 + 输出编码 |
| CSRF攻击 | Token校验 |

## 九、监控与运维

### 9.1 监控指标

- **APM**: Skywalking / Micrometer
- **Metrics**: Prometheus + Grafana
- **Logging**: ELK Stack
- **Tracing**: OpenTelemetry

### 9.2 告警策略

- 错误率 > 1% 触发告警
- 响应时间 P99 > 3s 触发告警
- CPU > 80% 触发告警
- 内存 > 85% 触发告警
- 数据库连接池 > 80% 触发告警
