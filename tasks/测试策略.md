## 1. 测试范围

本文档定义了 WMS 低代码平台 Phase 0 和 Phase 1 MVP 的完整测试策略。

## 2. 测试策略

### 2.1 测试分层

| 测试类型 | 范围 | 测试工具 |
|---------|------|---------|
| DDL 测试 | sys_column_meta 扩展 4 个字段 | SQL 验证 |
| 接口测试 | Meta Schema API + CRUD API | Postman / REST Client |
| 单元测试 | FieldRenderer、LowcodeForm 组件 | Vitest / Jest |
| E2E 测试 | WMS0010 仓库管理完整页面 | Playwright / Cypress |
| 集成测试 | LowcodeLayout + WmsSearchBar + LowcodeForm | Playwright |

### 2.2 测试环境

- **开发环境：** 本地 IDE
- **测试环境：** 测试服务器
- **预发布环境：** 预发布验证

## 4. 测试通过标准

- 所有 P0 用例 100% 通过
- P1 用例通过率 ≥ 90%
- 无阻塞性 Bug 遗留
